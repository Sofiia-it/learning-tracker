# Personal Learning Tracker

Personal learning tracker written in Python.

A simple command-line application written in Python that helps track daily learning activities.

## Features
- Add new learning activities
- Automatically saves the date
- View all saved activities with numbering
- Stores data in a text file

## Technologies
- Python 3
- File handling
- Command-line interface (CLI)

## How to run
```bash
python tracker.py
```
